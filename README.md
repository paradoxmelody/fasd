## 📖 Overview

This repository hosts the official website for Fetal Alcohol Spectrum Disorders (FASD) in South Africa. The platform serves as a vital resource, offering comprehensive information, educational materials, and support guidance to individuals, families, and communities impacted by FASD. Its primary goal is to raise awareness, promote understanding, and facilitate access to necessary support and health promotion initiatives within the South African context.

## ✨ Features

-   🎯 **Comprehensive Information**: Detailed pages covering various aspects of FASD, its prevention, diagnosis, and management.
-   📚 **Educational Resources**: Content tailored to inform about FASD health promotion.
-   👨‍👩‍👧‍👦 **Support for Families**: Guides and resources for living with and supporting a child or an individual with FASD.
-   ❓ **Frequently Asked Questions (Q&A)**: A dedicated section to address common queries about FASD.
-   📞 **Contact Information**: Easy access to contact details for support and inquiries.
-   🤝 **Task Team Overview**: Information about the individuals and organizations involved in the FASD South Africa initiative.

## 🚀 Quick Start

This project is a static website, meaning it consists of simple HTML and CSS files that can be viewed directly in a web browser without any server-side processing or complex build steps.

### Prerequisites
-   A modern web browser (e.g., Chrome, Firefox, Edge, Safari)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/paradoxmelody/fasd.git
    cd fasd
    ```

2.  **Open in your browser**
    Simply open the `index.html` file (or any other `.html` file) in your preferred web browser.
    ```bash
    # Example for Linux/macOS
    open index.html

    # Example for Windows
    start index.html
    ```

### Configuration Files

## 🚀 Deployment

This static website can be deployed easily on various static site hosting services.

### Deployment Options
-   **GitHub Pages**: As a static site hosted directly from a GitHub repository, it is ideal for GitHub Pages deployment. The live demo link currently points to a potential GitHub Pages URL.
-   **Netlify / Vercel**: Services like Netlify or Vercel offer straightforward deployment for static sites by connecting directly to your GitHub repository.
-   **Any Web Server**: The entire content of the repository can be uploaded to any traditional web server (Apache, Nginx, etc.) and served directly.
